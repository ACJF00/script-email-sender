sendergmail.py
This script sends an email using Gmail's SMTP server. It requires the sender's email address, receiver's email address, and the sender's password to authenticate with the SMTP server.

Usage
Replace the SENDER, RECEIVER, and PASSWORD variables with the appropriate values.
Replace the html variable with the HTML content of the email.
Run the script.
Dependencies
Python 3.x
smtplib module
ssl module
email.mime.text module
email.mime.multipart module
